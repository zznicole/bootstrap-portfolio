About

As a self-motivated front-end developer with UX/UI focus, I’m passionate about web development, as well as enjoying taking on coding challenges, solving problems, and learning new things during each project. With cross-industries experiences, I see each project from an idea till delivering to clients as a whole picture.

Previously graduated and worked as Graphic Designer, which enabled me to have a good eye in Visual and User Interface design. the study of User Experience design-related courses and years of experiences in hospitality, taught me how to emphasize our user’s needs to ensure better user experiences.

To satisfy my curiosity and keep my knowledge up to date, I regularly attend Tech or Design related events (Don't feel surprised if you find me there :) I’m also a member of Interaction Design Foundation Linkoping, Sweden.

Audio Transcription

A web application to be able to convert voice to text by simply click start button. Once the green start button is clicked, the application starts to record the audio and directly put into text in the screen section below the buttons. It can convert several lauguages at same time, regardless the writing system of the language.

I built and designed this web application in HTML5, CSS3,SASS, Vanilla Javascript, by using speech recognition api.

New Finder

A web application for users to search news which they are interested. Once the keywords which are related to the news are typed into the input bar, then click search button. Users will get a list of news are related to what they are searching for. News infos include picture, title, author, published time, day, and year, as well as the beginning part of content, user can dive into the whole content of the news by click on more info.

I built and designed this web application in HTML5, CSS3,SASS, Vue.js, by using news api.
